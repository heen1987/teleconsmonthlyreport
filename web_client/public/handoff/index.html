AI-PMS 파트별 확인 초안

Project_ID 중심 PMS 흐름, 담당 파트별 검토 항목, 외부 접속 URL, Android APK 제공 정보를 한 화면에서 확인하는 팀 공유용 페이지입니다.

공유 기준

현재 URL은 Cloudflare quick tunnel 기반 임시 주소입니다. Mac mini 또는 터널 세션이 재시작되면 주소가 바뀔 수 있습니다.

김강현

Collection API 확인

  • Android upload session, token, multipart upload 계약
  • worker heartbeat, claim, lease, retry 상태 전이
  • Platform callback signing, replay, backoff
  • 파일 검증, checksum, 저장 경로, 실패 케이스
박주연

Platform API 확인

  • auth, admin, project, meeting, approval 권한 경계
  • Project_ID 기준 task, decision, risk, resource, cost 연결
  • Collection callback idempotency와 schema validation
  • email distribution, ERP handoff, audit log
김희섭

Android/Web/통합 확인

  • 실기기 login, recording, upload, job status E2E
  • Web review, approval, distribution, operations UX
  • Mac mini audio job claim, STT/LLM 완료 검증
  • 임시 터널, APK, 고정 도메인 전환 필요성
